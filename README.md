# NewsDemo

News Demo is an app for view the most popular articles on NYTimes.com based on emails, shares or views.

Features:
 - View the news on list
 - View the new on Detail Screen
 - Add news to Favorites
 
MVVM+RxSwift arch, CoreData for saving.
